# Teslo API
1. Clonar el repositorio
2. ```bash
    $ npm install
    # or
    $ yarn install
    ```
3. Renombrar el archivo `.env.example` a `.env`
4. Cambiar los valores de las variables de entorno
5. Levantar la base de datos con 
   ```bash
    $ docker-compose up -d
    ```
6. Levantar en modo desarrollo con 
   ```bash
    $ npm run start:dev
    # or
    $ yarn start:dev
    ```
